#senior-project

A fuzzy control based single-axis solar tracker system.

Materials used:
* Arduino Uno
* DRV8825
* Stepper Motor (Nema 23)
* 2 light dependent resistor
* 2 resistors (10k Ohms)
* Electrolytic capacitor (25V, 1000uF)
